Primary comparisons

This is comparing two PNG files to ffmpeg converted versions of those files. Ideally you should be able to see rec2020 if your monitor is rec2020, and P3-D65 if you have a P3 display or will not see any text if nothing is displayed. Since most people do not have rec2020 displays, you shouldnt see the rec2020 text at all. The code to generate these files is here. You can reorder the images to help with comparison by drag and drop.

    • -color_primaries 12 = display p3 mp4

    P3-D65 should be visible only if you have a P3-D65 monitor, (unless the browser is broken).

    ffmpeg -loop 1 -y -i ps-combined-displayp3-g2.2.png -sws_flags spline+accurate_rnd+full_chroma_int -vf "scale=in_range=full:in_color_matrix=bt709:out_range=full:out_color_matrix=bt709" -c:v libx264 -t 10 -pix_fmt yuv420p -qscale:v 1 -color_range 2 -colorspace 1 -color_primaries 12 -color_trc 13 ./iccgamut/greyscale-displayp3.mp4
